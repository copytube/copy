붙여넣기 도움말
• URL 입력칸에 직접 붙여넣거나, 오른쪽 “클립보드 붙여넣기” 버튼을 사용하세요. (브라우저/OS 허용 필요)
• URL 다중등록: 입력칸에 한 줄에 1개의 URL을 넣고, 카테고리 선택 후 “등록”을 누르면 모두 등록됩니다.
• 위의 “아래부터/위부터 등록” 옵션으로 등록 순서를 정합니다. (피드는 최신순이라 시리즈는 보통 아래부터가 적합합니다.)

• HTTPS 환경에서만 작동합니다. (GitHub Pages OK)
• iPhone/iPad: 처음 붙여넣기 시 “이 앱에서 붙여넣기 허용” 팝업에서 허용 선택
• 안드로이드/데스크탑 Chrome: 설정 → 사이트 설정 → 클립보드에서 https://copytube.github.io 허용
• 또는 길게 누르기 → 붙여넣기 혹은 Ctrl/⌘+V를 사용하셔도 됩니다.

• 개인자료는 로컬저장소에 저장되며 공유되지 않습니다. 이름 변경 가능하고, 위치를 상단으로 올릴 수도 있습니다(아래 스위치).

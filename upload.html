▶ 붙여넣기 도움말
• URL 입력칸에 직접 붙여넣거나, 오른쪽 “클립보드 붙여넣기” 버튼을 사용하세요. (브라우저/OS 허용 필요)
• URL다중등록: URL 입력박스에 한 줄에 1개의 URL을 붙여넣고, 카테고리 선택 후 “등록”을 누르면 한 번에 모두 등록됩니다.
• 위의 “아래부터/위부터 등록” 옵션으로 등록 순서를 결정합니다. (피드는 최신순이라 시리즈는 보통 아래부터 등록해야 차례대로 보여집니다.)

• HTTPS 환경에서만 동작합니다. (GitHub Pages OK)
• iPhone/iPad (iOS 16+): 처음 붙여넣기 때 “이 앱에서 붙여넣기 허용” 팝업에서 허용을 선택하세요.
• 안드로이드/데스크탑 Chrome: 설정 → 사이트 설정 → 클립보드에서 https://copytube.github.io를 허용으로 변경하세요.
• 또는 길게 누르기 → 붙여넣기 또는 Ctrl/⌘+V를 쓰면 됩니다.

• 개인자료 1/2는 로컬저장소에 저장되며 공유되지 않습니다. 이름 변경 가능하며, 카테고리 위치(상단/하단)도 변경 가능합니다.

카테고리는 최대 3개까지 선택 가능합니다.
